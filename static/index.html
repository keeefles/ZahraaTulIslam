User Routing
Path HTTP methods Description
/users/ GET Display all users
/users/register POST Register a new user
/users/:id GET Retrieve one user
/delete/:id DELETE Delete user
/login/ GET Login to an account

Post Routing
Path HTTP methods Description
/posts/ GET Display all posts
/posts/addpost/ POST Add a new post
/posts/:id GET Retrieve one post
/delete/:id DELETE Delete post
